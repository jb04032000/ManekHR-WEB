# Textile-trade taxonomy (Zari360 Connect)

Exhaustive map of the Indian textile trade (Surat/Gujarat focus) — machines,
processes, materials/trims/chemicals, and job roles across the whole value chain.

**Why this exists:** built 2026-06-27 from a multi-agent value-chain sweep so the
trade's full vocabulary is captured in ONE place. A curated, high-value subset is
mirrored into the site's SEO keyword set (`components/marketing/schema.ts`
`SITE_KEYWORDS`), but the keyword tag should NOT hold all of this (keyword
stuffing). **The real home for this list is the app's category taxonomy** — the
Connect jobs board roles, the services directory, product/material tags, and
machine-marketplace categories. Each category becomes its own page that ranks,
which beats a long keyword tag. Use this doc to drive that.

Relevance tags: [H]igh / [M]ed / [L]ow search relevance for an India textile B2B network.
